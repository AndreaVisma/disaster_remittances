# Migrants as First Responders: A Global Estimate of Disaster-Driven Remittances
[![DOI](https://zenodo.org/badge/1118835193.svg)](https://doi.org/10.5281/zenodo.18466024)
This repository contains data and replication code for Vismara et al. (forthcoming). "Migrants as First Responders: A Global Estimate of Disaster-Driven Remittances". 

The pre-print version of the paper is available on [arXiv](https://arxiv.org/abs/2512.16373), data and code will be made available upon publication.
